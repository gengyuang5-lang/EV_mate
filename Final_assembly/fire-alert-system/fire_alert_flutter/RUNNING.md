## ⚠️ 注意事项

1. **后端服务**: 确保后端服务运行在 http://localhost:3000
2. **模拟器网络**: 如果使用Android模拟器，可能需要将 `localhost` 改为 `10.0.2.2`
3. **权限**: 应用需要位置权限才能使用逃生路线功能

## 🔍 检查后端服务

如果应用无法连接后端，请启动后端服务：

```bash
cd ../backend
npm start
```
