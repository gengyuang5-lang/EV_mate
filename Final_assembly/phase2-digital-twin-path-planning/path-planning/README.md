# AI 路径规划模块

## 功能概述

实现智能路径规划系统，包括基础路径规划和弱势群体适配功能。

## 核心功能

### 1. 基础路径规划
- 火点触发后自动生成逃生路径
- 避开火点蔓延区域和拥堵路段
- 动态路径更新（每 500 毫秒）

### 2. 弱势群体适配
- 老人/儿童/残障人士模式
- 优先选择无障碍通道
- 避开陡坡/窄道
- 引导速度降低 30%
- 推送协助信号

## 模块结构

### core/ - 核心路径规划算法
- A* 算法实现
- 路径优化算法

### agent/ - AI Agent
- 智能决策逻辑
- 环境感知

### dynamic-updater/ - 动态路径更新
- 500ms 更新机制
- 实时路径调整

### vulnerable-group/ - 弱势群体适配
- 无障碍路径规划
- 速度调整
- 协助信号推送

### obstacle-avoidance/ - 障碍物避让
- 火点避让
- 拥堵路段避让

## 性能指标

- 响应时间：< 1 秒
- 动态调整准确率：> 90%
- 无障碍标准合规性：100%
