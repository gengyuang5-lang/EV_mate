# 测试模块

## 测试目标

### 性能指标
- **路径规划响应时间**：< 1 秒
- **动态调整准确率**：> 90%
- **弱势群体路径适配**：符合建筑无障碍设计标准

## 测试内容

### 1. 响应时间测试
- 测试路径规划算法的响应时间
- 验证是否满足 < 1 秒的要求

### 2. 路径准确性测试
- 测试路径的有效性
- 验证路径是否避开障碍物
- 验证路径是否到达目标出口

### 3. 动态调整测试
- 测试动态路径更新的准确性
- 验证环境变化时的路径调整能力
- 验证准确率是否 > 90%

### 4. 无障碍标准合规性测试
- 测试弱势群体路径是否符合无障碍设计标准
- 验证通道宽度、坡度等指标

## 运行测试

```bash
# 运行所有测试
npm test

# 运行特定测试
npm test -- response-time
npm test -- accuracy
npm test -- accessibility
```
